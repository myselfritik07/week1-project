# Week 1 Portfolio

A simple personal portfolio website built with HTML, CSS, and JavaScript.

## What is in this folder
- index.html — main page
- css/style.css — styling
- js/script.js — small interactions
- assets/images/ — images for the site

## How to view it
Open index.html in your browser, or use Live Server in VS Code.

## How to customize it
- Edit the text and links in index.html
- Update the design in css/style.css
- Replace images in assets/images/

## For GitHub Pages
If you want to publish it online, upload the folder to a GitHub repository and enable Pages from the repository settings.
