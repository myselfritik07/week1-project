### Next Steps to Deploy

**Step 1: Create GitHub Repository**
1. Go to https://github.com/new
2. Enter `week1-portfolio` as repository name
3. Make it Public
4. Click "Create repository"

**Step 2: Push to GitHub**
```bash
cd "c:\Users\akhil\OneDrive\Desktop\week 1\week1-portfolio"
git remote add origin https://github.com/yourusername/week1-portfolio.git
git branch -M main
git push -u origin main
```

**Step 3: Enable GitHub Pages**
1. Go to Settings → Pages
2. Select "main" branch as source
3. Save
4. Your site will be live at: `https://yourusername.github.io/week1-portfolio`

**Step 4: Add Your Images**
1. Create images and place them in `assets/images/`
2. Commit and push: `git add . && git commit -m "Add images" && git push`
